# ayunchatgptcord
ChatGPT as a Discord server!

### Usage:
```batch
@echo off
set "BOT_TOKEN="
set "ENDPOINT=https://api.openai.com/v1/chat/completions"
set "APIKEY="
set "ENDPOINT_GPT4=https://api.openai.com/v1/chat/completions"
set "APIKEY_GPT4="
set "INACTIVE_TIME=10m"
set "MAX_TOKENS=256"
set "MAX_THREADS=3"
set "INFO_ENDPOINT="
set "INFO_APIKEY="
set "ENABLE_GPT4=true"
set "MAX_MSGS=10"
go run .
```
